<h2>Start the sound driver on the raspberry pi :</h2>
sudo modprobe snd_bcm2835<br><br>

<h2>make sure that snd_bcm2835 is running by :</h2>
lsmod<br><br>

<h2>Run the player by :</h2>
node app.js<br><br>
