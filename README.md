<b>Start the sound driver on the raspberry pi :</b>
sudo modprobe snd_bcm2835<br><br>

<b>make sure that snd_bcm2835 is running by :</b>
lsmod<br><br>

<b>Run the player by :</b>
node app.js<br><br>
